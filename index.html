Documentation Breadtime

1. Règles/version

Version du serveur 1.21.7 (Paper).
Adresse IP : breadtime.omgcraft.fr

Bien évidemment, pas de grief des constructions des autres, soyez respectueux, pas de cheat, etc.
(Soyez pas con quoi).

Le thème du spawn n’est pas un “thème” mais impose l’utilisation en majorité d’un bloc et ses variantes. Ce bloc est le PALE OAK.

2. Datapacks

  • Terralith : Permet une génération customisée du monde.

  • Structory : Permet la génération de nouvelles structures dans le monde.

  • Dungeons and Taverns : Permet la génération de nouvelles structures dans le monde.

  • Towns and Towers : Permet la génération de nouvelles structures dans le monde.

  • Nullscape : Modifie la génération de l’End.

  • Waystones : Permet de se téléporter d’un point A à un point B à l’aide de pilier physique.
    Les différents types de blocs dans le craft : Andesite, Sandstone, Nether Bricks, Mossy Stone Bricks, Cobbled Deepslate, Copper Block, Prismarine Bricks, Tuff.

  • Blaze and Cave’s Advancement : Rajoute une grande variété d’advancements.

  • Armor Statues : Permet de changer les poses des armors stands à l’aide d’un livre (peut-être donné par Vacheuh).

  • Armored Elytra : Permet de « fusionner » des Élytres et un plastron.

    • Enchantements conservés.
    • Pour fusionner : lancer une Élytre et un plastron au-dessus d’une enclume.
    • Pour séparer : lancer l’Élytre sur un Grindstone.

  • Elevators :

    • Drop une Ender Pearl sur une laine colorée.
    • Sauter pour monter, s’accroupir pour descendre.
    • Nécessite un second bloc relié.

  • Mini blocks : Utiliser un stone cutter pour obtenir une version mini du bloc.

  • Graves : Une tombe apparaît à la mort d’un joueur contenant son stuff.

  • Cauldron Concrete : Lancer de la poudre de béton dans un chaudron la transforme en béton.

  • Cauldron Mud : Lancer de la terre dans un chaudron la transforme en boue.

  • Chunkloaders : Drop une Nether Star sur une lodestone (magnétite) permet de créer un chunk loader.

  • Double Shulker Shells : Améliore le drop des shulker shells

  • Painting Picker : Permet d’utiliser un stonecutter pour choisir une peinture.

  • Better Item Frame :

    • Sneak + clic droit avec des cisailles : rend invisibles.
    • Sneak + clic droit avec un pinceau : rend visibles.
    • Sneak + clic droit avec une vitre : bloque la rotation.

  • Silk Touch Budding Amethyst : Permet de récupérer les buddings amethyst avec silk touch.

  • Storage Pots: Permet le stockage de masse d’un item (256 stacks).

    Pour voir le contenu exact du stockage il faut utiliser ceci (peu importe le morceau de vase) :

  • Wandering Trades : Les wandering traders peuvent maintenant vendre des mini-blocs.

3. Modification par rapport à Minecraft Classique

  • Ghast et Creeper ne causent pas de grief.
  • 30% des joueurs connectés suffisent pour passer la nuit.
  • Compteur de morts.
  • Jusqu’à 130 créatures sur un même bloc sans suffocation.
  • Pas de “location bar”
  • Lorsqu’on tombe dans le vide dans l’End, on est téleporté dans l’overworld.

4. Vacheuh

  • En échange d’un service rendu à Vacheuh, vous pouvez colorer votre pseudo avec n’importe quelle couleur, y compris hexadécimale.
  • Des bonus peuvent être débloqués au fur et à mesure avec Vacheuh.
  • Si vous lisez ceci, envoyez “♥” à Vacheuh. Vous aurez ptet droit à un kdo.

5. Évènements aléatoires

⚠️ Conseil : mettez-vous en /afk si vous vous absentez pour éviter une mort bête !
Vous pouvez également demander à ne pas être concerné par ces évènements, il faut en notifier ByNetix.
Attention : il ne faudra pas utiliser la commande /afk

Des évènements peuvent se déclencher de façon aléatoire (ou non) :

  • CatAllergy : Approcher un chat → Poison II 5s
  • WolfAllergy : Approcher un loup → Poison II 5s
  • BeeAllergy : Approcher une abeille → Poison I 5s + Nausée II 10s
  • OreExplosion : Miner un minerai → 1 chance sur 5 qu’il explose (2,5 cœurs de dégâts min.)
  • ZombieHorde : Horde de 8–17 zombies sur un joueur aléatoire (sauf AFK)
  • PillagerHorde : 8–13 pillards sur un joueur aléatoire (sauf AFK)
  • GhostInvasion : 50% des joueurs attaqués par 4 fantômes (sauf AFK)
  • EndermiteInvasion : 10–15 endermites spawnent sous tous les joueurs (sauf AFK)
  • DivineBlessing :
    Tous les joueurs reçoivent :
    • Vitesse II, Résistance II, Célérité III, Force II, Résistance feu (3 min)
    • +10 cœurs (3 min)
    • Régénération II (1 min)
  • Famine : Faim II (3 min) + perte possible d’un slot de nourriture (hors pommes dorées)
  • ReverseGravity : Gravité inversée.

    Si vous ne bougez pas, vous ne mourrez pas… normalement.

  • AcidRain : Pluie acide qui inflige des dégâts aux joueurs non abrités.
  • BloodMoon : Toutes les créatures sont renforcées pendant la nuit.
  • Boss : Un boss apparaît à des coordonnées précises avec loot spécial.
  • NoRegen : Régénération naturelle désactivée pendant 20 min.
  • KeepInventory : Aucun drop d’inventaire à la mort pendant 20 min.
  • PolarClimate : Le froid empêche les cultures de pousser.
  • ToxicWater : Eau devient toxique → Poison II 3s + Nausée I 10s
  • NetherInvasion : Monstres du Nether attaquent un joueur (sauf AFK)
  • IllagerInvasion : Horde d’Illageois attaque un joueur (sauf AFK)
  • TreasureHunt : Un coffre apparaît, indices fournis.
  • AmIFast : Donne un effet de célérité assez puissant.

La production dément tout recyclage.
